# 🚀 Power Platform Portfolio

## 👨‍💻 Sobre mim

Atuo com desenvolvimento de soluções utilizando Power Platform (Power Apps e Power Automate), com foco em automação de processos internos, padronização de fluxos e redução de retrabalho.

Experiência prática na construção de aplicações corporativas integradas com SharePoint, incluindo controle de solicitações, fluxos de aprovação e gestão de dados.

---

# 📌 Principais Soluções Desenvolvidas

## 🔷 Portal Corporativo de Aplicações

Desenvolvimento de um portal centralizado no Power Apps para acesso a múltiplas soluções internas.

### 🧩 Funcionalidades

* Centralização de sistemas internos
* Navegação por categorias (Desenvolvimento, Administrativo, Apoio)
* Acesso rápido a aplicações e fluxos
* Interface amigável para usuários finais

---

## 🔷 Sistema de Nota de Alteração (Firmware / Hardware)

Aplicação completa para gerenciamento de solicitações de alteração técnica.

### ⚙️ Recursos implementados

* Criação de solicitações com status controlado
* Geração de identificador único
* Edição e continuidade de solicitações (modo rascunho)
* Integração com SharePoint
* Fluxo de aprovação automatizado via Power Automate

### 🔄 Automação

* Criação automática de registros de aprovação
* Controle de múltiplos aprovadores
* Atualização de status em tempo real
* Notificações automáticas

---

## 🔷 Fluxos de Aprovação Automatizados

Criação de fluxos complexos no Power Automate para diferentes processos internos.

### 🧠 Lógicas aplicadas

* Condições por tipo de solicitação
* Controle de múltiplos aprovadores (Apply to each)
* Prevenção de duplicidade
* Validação de dados antes de aprovar

---

## 🔷 Automação de Anexos (SharePoint)

Gerenciamento automatizado de anexos entre listas.

### ⚙️ Funcionalidades

* Coleta automática de anexos
* Associação com registros corretos
* Processamento em lote
* Organização por tipo de solicitação

---

## 🔷 Sistema de Apontamento de Horas

Aplicação para controle de horas em projetos.

### 🧩 Funcionalidades

* Registro de horas por projeto
* Organização por ano
* Gestão de recursos
* Interface simples para usuários

---

# 🛠️ Tecnologias Utilizadas

* Power Apps (Canvas Apps)
* Power Automate
* SharePoint
* Integrações com Outlook

---

# 🧠 Diferenciais Técnicos

* Controle de identificadores personalizados
* Estruturação de fluxos complexos de aprovação
* Tratamento de duplicidade de dados
* Uso avançado de variáveis e contexto no Power Apps
* Organização de múltiplos fluxos dentro de soluções corporativas
